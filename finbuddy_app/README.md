# FinBuddy — Finance Dashboard (Streamlit Cloud)

Upload bank CSV files, analyze expenses, view charts, and chat with your transactions.

### Features
- CSV parsing (robust)
- Merchant extraction
- Category tagging
- Monthly charts
- Category donut
- Top merchants
- Simple chatbot
- Works on Streamlit Cloud

### Deployment
1. Push this repo to GitHub  
2. Go to https://share.streamlit.io  
3. Deploy → choose this repo → select `streamlit_app.py`  
4. Add `OPENAI_API_KEY` under Secrets if you want chat to use OpenAI
